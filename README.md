# 📸 Hexula Photo Gallery

A modern and responsive photo gallery built with **HTML, CSS, and JavaScript**. Hexula Photo Gallery provides an elegant browsing experience with curated image collections, category-based filtering, smooth animations, and a **full-scale image preview** for immersive viewing.

## ✨ Features

* 📷 Responsive gallery layout
* 🗂️ Category-based image filtering
* 🖼️ Full-scale image preview
* ⚡ Smooth animations and transitions
* 📱 Fully responsive across desktop, tablet, and mobile devices
* 🎨 Clean and modern user interface
* 🌄 High-quality image collections
* 🚀 Lightweight and fast-loading

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Postimg (Image Hosting)
